Installation

Get started with Wolf Parallax in 4 easy steps!

Step 1 - Include CSS

Add the wolf css file to the <head> of your document.

<link rel="stylesheet" href="files/css/wolf.css" />

Step 2 - Include JS

Add the javascript/jQuery files in the footer of your document. You can also choose to host jQuery on your own server, but Google is nice enough to take care of that for us.

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script>
<script type='text/javascript' src='tweenMax.js'></script>
<script type='text/javascript' src='jquery.wolf.min.js'></script>

Step 3 - Add HTML markup

Make sure to use this html markup for your. Start with a single container element
<div class="wolf-grid">
Continue to add your different items. It is important to use classes wolf-item and wolf-item-inner and wolf-media because the script targets them specifically. Place your images and anything else you desire into each wolf-item-inner. 

<div class="wolf-grid">

  <div class="wolf-item">
    <div class="wolf-item-inner">
      <div class="wolf-media"><img src="image1.jpg"></div>
      <div class="wolf-caption">caption content 1</div>
    </div>
  </div>
  
  <div class="wolf-item">
    <div class="wolf-item-inner">
      <div class="wolf-media"><img src="image2.jpg"></div>
      <div class="wolf-caption">caption content 2</div>
    </div>
  </div>
  
  <div class="wolf-item">
    <div class="wolf-item-inner">
      <div class="wolf-media"><img src="image3.jpg"></div>
      <div class="wolf-caption">caption content 3</div>
    </div>
  </div>
  
</div>

You can use custom names for wolf-grid

Do not change the classes wolf-item and wolf-item-inner and wolf-media

Step 4 - Call the wolf script

Lastly, call the wolf initialiser on your javascript file or in the <head> of your document.

jQuery(window).load(function($) {
  jQuery('.wolf-grid').wolf();
});

Options

See here the list of all script options which you can call by initialising the plugin

parallax

Type: boolean
Default: true

Adds/Remove the default parallax effect on scroll.

captionparallax

Type: boolean
Default: true

When true the caption will have an additional parallax effect relative to the wolf-media element. This only have impact if you use wolf-caption class in your items.

speed

Type: number
Default: 1

The default speed will be multiplicated by this value.

randomhorizontaloffset

Type: boolean
Default: false

Adds a horizontal random offset for each item

randomverticaloffset

Type: boolean
Default: false

Adds a vertical random offset for each item

mouseparallax

Type: boolean
Default: false

Adds an additional parallax effect on mouse move

filter

Type: string (classname of item)
Default: false

Filters the items with the given classname. Do not call this option on initialise.

filtertype

Type: string
Default: hide

Filter type. Choose between hide and opacity



Example:

jQuery('.wolf-grid').wolf({
  parallax:false,
  randomhorizontaloffset:true
});

You can also call the options via the data attribute on the wolf container element. Note that this will overwrite the options called on the initialiser via javascript.

<div class="wolf-grid" data-randomhorizontaloffset="true" data-mouseparallax="true">

Item size

By default the script will align 2 items per row. ( half / half of the parent container width)

You have the option to individually add rows with 3items or 1 single items. Add the class wfull or wthird to the wolf-item.

<div class="wolf-item wfull">
or
<div class="wolf-item wthird">

Custom Speed

You have the possibility to change the speed of each item individually. By default the script adds random parallax speeds (on scroll) to the different items. To change/overwrite them you just need to add the data-speed attribute on the wanted item.
Choose any positiv or negativ value.

<div class="wolf-item" data-speed="2">
or
<div class="wolf-item" data-speed="-0.8">

The higher the spread is between the different speeds, the more chance there is that some items will overlay others!
So it's recommended to choose a value between -2 and +2 but it's up to you.

Caption options

If you don't add any classes to the wolf-caption, the captions are postioned by default:
- on the top right corner for all image elements on the left site,
- and on the bottom left corner for all image elements on the right site

To control the position of your captions you can add the classes to the wolf-caption element:
- 'ctop' OR 'cbottom' AND
- 'cleft' OR 'cright' OR 'ccenter'
to change the position individually for each wolf-caption element 

<div class="wolf-caption ctop cright">

Note: If you add these classes, it will force the position, even after you filtered the items which could lead to some display issues.
If you don't have specified the positioning by the above classes and the filtertype 'hide' is active, the position of the caption will change depending the image position (mirror effect).

Adding additional offset to the captions

Additionally you can add some offset values using inline css to have even more control on it.

use margin-right if your caption has right position relative to the media and margin-left if it has left position

<!-- for right positioned captions relative to the media -->                
<div class="wolf-caption" style="margin-right:50px">

<!-- for left positioned captions relative to the media -->                
<div class="wolf-caption" style="margin-left:20%">

use margin-top if your caption has top position relative to the image and margin-bottom if it has bottom position

<!-- for top positioned captions relative to the media -->                
<div class="wolf-caption" style="margin-top:20%">

<!-- for bottom positioned captions relative to the media -->                
<div class="wolf-caption" style="margin-bottom:40px">

ATTENTION: All your custom caption position settings/options don't have effect on mobile devices where the parallax is not supported.

Static captions

As described above, the captions will position top/right or bottom/left by default. By adding the class static to the caption element, it will delete the position and just be placed below the media element without any styling.

<div class="wolf-caption static">

Text captions

The captions should only contain some short infos like a name, title, etc. If you're planning to add more text it's recommended to use the class wolf-text to the wolf-item. It uses a different approach to place the caption which is important for responsive reasons.

Optionally you can also add a custom width in order to have control how much the caption is overlaying the media element or not. it's recommended to use % values for responsive.

<div class="wolf-item wolf-text" style="width:50%">

Filter

Wolf can hide and show items with the filter option. Items that match that filter will be shown. Items that do not match will be hidden.

Add classes to items

Use class selectors to easily filter items.
For example, each item can have several identifying classes: web, photography, print, branding.

<div class="wolf-item web print">...

<div class="wolf-item branding">...

<div class="wolf-item photgraphy">...

<div class="wolf-item print branding">...

Filter option

Set a selector with the filter option on a click event.

// filter .web items             
jQuery('.wolf-grid').wolf({ filter: '.web', filtertype: 'hide' });

// filter .photography items with opacity type             
jQuery('.wolf-grid').wolf({ filter: '.photography', filtertype: 'opacity' });
